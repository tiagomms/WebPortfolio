In April 2017 I began applying for jobs as a frontend developer. Due to not having much experience in Web Programming and having been asked by some companies to provide a portfolio, I decided to create my own web portfolio in the beginning of May.

The goal of this portfolio is to display my studies and projects I have worked on in an objective manner, and show the relationships between them and the skills I have developed through time. The timespan is from September 2009, when I started my BSc degree in IST Lisbon, until August 2017.

*At the moment, CSS Grid Layouts are not compatible with IE or Microsoft Edge. I began designing a website mockup using Inkscape. In the development stage, I used the latest grid layouts on CSS3, Angular2 as the frontend framework and D3 to code each chart type. I used Vim as a text editor.

All design, code and text were carried out by me. I thank my sister, Catarina Silva, for reviewing the text and Miguel Tavares (@mgontav) for his advice during the website development. Throughout this process, I was inspired by Edward Tufte’s books: Envisioning Information (1990) and Visual Explanations: Images and Quantities, Evidence and Narrative (1997).
